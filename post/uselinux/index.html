查看资源使用情况：

free -m -s5 —查看内存【-s5：5秒自动更新显示】 （[－b　－k　－m] [－o] [－s delay] [－t] [－V]）

uptime —查看系统负载情况

df -h —查看磁盘（df [-akmhHTi]）

SecureCRTPortable 中文显示配置：选项—会话选项—终端—外观—字符编码—UTF-8

tail -10f log.txt —输出log.txt文件最后10行 参考：https://ruby-china.org/topics/20154

rm -rf dirname —删除非空的目录 (-i 删除前逐一询问确认。-f 即使原档案属性设为唯读亦直接删除无需逐一确认。 -r 将目录及以下之档案亦逐一删除 )

nohup java -jar springboot-helloworld-0.0.1-SNAPSHOT.jar & —后台运行(默认会产生nohup.out文件，可以通过>log.txt来指定控制台输出)

wget -c –limit-rate=3m http://x.x.com/a.zip — 下载【-c：断点续传；–limit-rate：限速(k、m)】

ps -ef | grep psName| grep -v grep | awk ‘{print $2}’ | xargs kill -9 —根据进程名称关闭进程

Linux下如何在不解压jar包查看或修改配置文件yum install telnet —安装telnet客户端—————————————————————————– netstat -lnp|grep 80 —查看端口占用情况

ps -aux |grep tomcat —查看tomcat进程

ps -ef|grep java|grep -v grep —显示出所有的java进程，去处掉当前的grep进程。

参考：https://blog.csdn.net/r13929847477/article/details/52336925

systemctl list-unit-files –type service —查看所有服务的开机自启状态

systemctl is-active name.service —查看某服务当前激活与否的状态

systemctl start/stop/restart name.service —启动/停止/重启某服务
