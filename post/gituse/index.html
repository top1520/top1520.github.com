Git使用记录

官网     官网下载页     淘宝nmp下载页

克隆仓库源码：git clone https://github.com/top1520/top1520.github.com.git

添加所有文件/目录：git add -A 填写提交信息：git commit -m “first commit” 提交到仓库：git push -u origin master

设置全局用户名：git config –global user.name “your name” 设置全局用户邮件：git config –global user.email “youremail@xxxmail.xxx”

设置当前仓库用户名：git config user.name “your name” 设置当前仓库用户邮件：git config user.email “youremail@xxxmail.xxx”

查看系统配置信息：git config –system –list 查看当前用户（global）配置：git config –global –list 查看当前仓库配置信息：git config –local –list
